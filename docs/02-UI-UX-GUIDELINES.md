## Design Principles

### 1. Clarity First
- Clear visual hierarchy
- Obvious call-to-actions
- Minimal cognitive load
- Consistent patterns

### 2. Efficient for Daily Use
- Quick access to frequent actions
- Keyboard shortcuts
- Smart defaults
- Batch operations support

### 3. Role-Appropriate
- UI adapts to user role
- Show only relevant options
- Appropriate complexity levels
- Age-appropriate for students

### 4. Trustworthy & Professional
- Clean, modern aesthetic
- Consistent branding
- Reliable feedback
- Error prevention

### 5. AI-Enhanced, Not AI-Dependent
- AI features enhance, not replace
- Clear AI vs human indicators
- Confidence levels shown
- Easy human override

## Typography

### Font Family

```css
/* Primary font - Inter */
--font-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* Monospace - for code */
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```

### Type Scale

| Name | Size | Line Height | Weight | Usage |
|------|------|-------------|--------|-------|
| Display | 36px / 2.25rem | 1.2 | 700 | Hero sections |
| H1 | 30px / 1.875rem | 1.3 | 700 | Page titles |
| H2 | 24px / 1.5rem | 1.35 | 600 | Section headers |
| H3 | 20px / 1.25rem | 1.4 | 600 | Card titles |
| H4 | 18px / 1.125rem | 1.4 | 600 | Subsections |
| Body Large | 18px / 1.125rem | 1.6 | 400 | Lead text |
| Body | 16px / 1rem | 1.5 | 400 | Default text |
| Body Small | 14px / 0.875rem | 1.5 | 400 | Secondary text |
| Caption | 12px / 0.75rem | 1.4 | 400 | Labels, captions |
| Tiny | 10px / 0.625rem | 1.4 | 500 | Badges |

1. **Sidebar**: Collapses to hamburger menu
2. **Tables**: Convert to card layout
3. **Multi-column**: Stack vertically
4. **Actions**: Move to bottom sheet or FAB
5. **Modals**: Full screen on mobile

## Animation Guidelines

### Timing

| Type | Duration | Easing |
|------|----------|--------|
| Micro (hover, focus) | 150ms | ease-out |
| Small (dropdowns) | 200ms | ease-out |
| Medium (modals) | 300ms | ease-in-out |
| Large (page transitions) | 400ms | ease-in-out |

### Motion Principles

1. **Purposeful**: Animation should guide, not distract
2. **Quick**: Users shouldn't wait for animations
3. **Natural**: Follow natural physics
4. **Consistent**: Same action = same animation

### Tailwind Animation Config

```javascript

  theme: {
    extend: {
      animation: {
        'fade-in': 'fadeIn 200ms ease-out',

        'spin-slow': 'spin 2s linear infinite',

          '0%': { opacity: '0' },

        },
```
